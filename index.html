// projeto 01

Manutenção Preditiva Industrial

Manufatura · Classificação · Detecção de Falhas

Modelo de classificação para prever falha de equipamento industrial antes que ocorra, usando parâmetros operacionais em tempo real: temperatura, rotação, torque e desgaste de ferramenta. Dataset real com 10.000 registros (UCI Machine Learning Repository). O threshold de decisão foi calibrado para priorizar detecção de falhas — decisão justificada pelo custo diferencial entre parada não planejada e inspeção preventiva.

0.964 AUC-ROC
68% Recall — falhas detectadas
1% Taxa de alarmes falsos
+58% Melhora no recall
Random Forest Classificação Binária Threshold Tuning Curva ROC Class Weight Matriz de Confusão Desbalanceamento de Classes

// projeto 02

Previsão de Consumo Energético Industrial

Siderurgia · Séries Temporais · Impacto Financeiro

Modelo de regressão para prever o consumo elétrico dos próximos 15 minutos em uma usina siderúrgica real (DAEWOO Steel Co., Coreia do Sul), viabilizando deslocamento de carga antes do horário de ponta — período com tarifa até 3x maior. Data leakage identificado e corrigido durante a validação. Economia estimada calculada com tarifas públicas da ANEEL para consumidor industrial Grupo A.

0.885 R² (modelo corrigido)
6.69 kWh Erro médio (MAE)
R$ 9.170 Economia anual estimada
35.040 Registros analisados
Random Forest Regressor Séries Temporais Feature Engineering Lag Features Data Leakage Divisão Temporal Tarifas ANEEL
